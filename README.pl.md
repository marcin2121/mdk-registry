# MDK Remote Registry

Zdalny rejestr komponentów i szablonów dla kreatora **MDK Setup Wizard**.

## 📂 Struktura katalogów

- **`components/`** – surowe pliki `.txt` z modułami React/Next.js (np. Bento Grid, FAQ).
- **`templates/`** – surowe pliki `.txt` z gotowymi stronami Landing Page (np. SaaS, Agency).

---

> [!NOTE]
> Repozytorium służy wyłącznie jako zdalna baza surowych plików dla skryptów instalacyjnych MDK. Pliki są przechowywane w formacie `.txt`, aby unikać błędów kompilacji/brakujących zależności w tym miejscu i gwarantować czyste wstrzykiwanie do docelowych solucji.
