# MDK Remote Registry

Remote database containing components and templates for the **MDK Setup Wizard**.

## 📂 Directory Structure

- **`components/`** – raw `.txt` files containing React/Next.js modules (e.g., Bento Grid, FAQ).
- **`templates/`** – raw `.txt` files containing full-sized Landing Page setups (e.g., SaaS, Agency).

---

> [!NOTE]
> This repository acts strictly as a remote code host for MDK setup tools. Files are deliberately stored with a `.txt` extension to avoid compilation errors/missing package warnings in this codebase, and to facilitate raw string fetching via API calls.
